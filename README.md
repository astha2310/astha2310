## 🚀 Featured Projects

### 🧠 [BehaviorLens](https://github.com/astha2310/BehaviorLens)
Multimodal behavioral AI platform using a tri-stream LSTM architecture fusing app-usage time-series, BERT text embeddings, and MobileNetV2 visual features. Outputs a real-time behavioral risk score with SHAP explainability. React dashboard with WebSocket live feed.
`PyTorch` `BERT` `LSTM` `React` `FastAPI` `SHAP`

### 🛡 SIEM-AI
Cloud-native Security Information & Event Management platform with AI-powered anomaly detection. Ingests logs from AWS CloudTrail, Azure Monitor, and Splunk HEC. PyTorch anomaly detection engine with a React dashboard.
`PyTorch` `FastAPI` `React` `AWS` `Docker`

### 🔍 NetGuard-IDS
ML-based network intrusion detection system operating at the packet level. Ingests Wireshark PCAP captures and Zeek logs, classifies threats with a PyTorch model.
`PyTorch` `Wireshark` `Zeek` `FastAPI` `Docker`

### ☁️ CloudGuard
Automated compliance monitoring for AWS cloud infrastructure against CIS Benchmark controls. Automates CIS Level 1 & 2 checks across IAM, S3, CloudTrail, VPC, and EC2.
`Python` `Terraform` `FastAPI` `React` `AWS`
